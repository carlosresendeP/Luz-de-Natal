# Tarefa
Crie um site de página única (Landing Page) responsivo, moderno e visualmente encantador, focado em exibir mensagens motivadoras de Natal e sobre o Nascimento de Jesus. O site deve ser desenvolvido utilizando HTML5, (pode usar Tailwind CSS via CDN) e JavaScript puro.

# Requisitos de Design e Estética
Paleta de Cores: Vermelho cereja, dourado suave, branco neve e verde pinheiro profundo.

Atmosfera: Utilize um efeito de "neve caindo" suave no fundo (via JS ou CSS).

Tipografia: Combine uma fonte elegante e cursiva para títulos (estilo "Handwriting") com uma fonte sans-serif limpa para as mensagens.

Elementos Visuais: Use cards com bordas arredondadas e sombras suaves (glassmorphism) para destacar as mensagens.

# Funcionalidades Necessárias
Gerador de Mensagens: Um botão central estilizado (ex: "Abrir Presente" ou "Nova Inspiração") que, ao ser clicado, exibe uma nova mensagem motivadora com uma transição suave (fade-in).

Lista de Mensagens: O sistema deve conter um array de pelo menos 10 mensagens que misturem o espírito natalino com temas de superação, gratidão e novos começos.

Botão de Compartilhamento: Um botão simples para "Copiar Mensagem" para que o usuário possa enviar para amigos.

Trilha Sonora (Opcional/Simulada): Um botão de "Mute/Unmute" para um som de lareira crepitando ou sinos suaves (apenas o ícone e a lógica, sem necessidade de arquivo externo pesado).

# Estrutura do Conteúdo
Header: Título impactante como "Luz de Natal: Pequenas Pílulas de Esperança".

Main: Área de exibição da mensagem atual.

Footer: Uma frase de encerramento desejando boas festas e um link fictício.
